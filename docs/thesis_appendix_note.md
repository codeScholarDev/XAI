# Appendix A: Optional Literature Classification Support Tool

An optional Python-based literature classification support tool was created to assist with organizing notes for the thesis titled:

**A Literature-Based Analysis of Explainable AI in Modern Intelligent Systems: Opportunities, Challenges, and Practical Implications**

## Purpose of the Tool

The purpose of the tool is to organize literature-review notes related to Explainable AI (XAI), classify them into broad thematic categories, and generate simple summary outputs that may support thesis planning and documentation.

## Input Data

The tool uses a CSV file containing source identifiers, author-year labels, titles, source types, short notes or abstracts, domains, and relevance notes.

## Classification Themes

The classification themes are:

1. Conceptual Foundations
2. XAI Methods
3. Opportunities
4. Challenges and Limitations
5. Human-Centered XAI
6. Practical Implications
7. Governance and Trustworthy AI

## Generated Outputs

The tool generates:

- A processed source CSV file.
- A source classification CSV file.
- A theme-wise Markdown summary.
- A Markdown literature report for documentation.

## Limitations

The tool uses rule-based keyword classification in mock mode and optional short OpenAI API calls in OpenAI mode. It does not replace manual reading, academic judgment, quality assessment, or the official literature review methodology.

## Academic Integrity Note

The tool is only an organizational support artefact. The official thesis methodology remains a literature-based review. All generated outputs should be manually reviewed, edited where necessary, and connected to properly cited academic sources before any thesis use.
