# Methodology Note

The thesis methodology is literature-based. The official research approach remains a structured/narrative literature review with systematic search and selection elements.

This software project is not the thesis methodology itself. It is an optional supporting artefact that helps organize literature-review notes after sources have been identified and selected. The tool can classify notes into broad themes, generate simple summaries, and produce documentation outputs for review.

The classification results must be checked manually. They do not replace academic database searching, source screening, quality assessment, citation management, or critical reading of the original publications.

The tool may be useful for planning chapter structure, identifying repeated topics, and preparing appendix material, but all thesis arguments and final interpretations should be written by the student and supported by properly cited academic sources.
